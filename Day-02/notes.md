# Day 2 - AWS Security Group Creation

## Task

Created a security group named `devops-sg` in the default VPC.

## Inbound Rules

* SSH (TCP 22) from `0.0.0.0/0`
* HTTP (TCP 80) from `0.0.0.0/0`

## Learning Outcomes

* Understood the purpose of Security Groups in AWS.
* Configured inbound traffic rules.
* Learned the difference between Security Groups and Network ACLs.
* Practiced working with resources inside the default VPC.
